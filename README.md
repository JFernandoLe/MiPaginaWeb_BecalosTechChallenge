# Mi Proyecto Final de HTML

### Descripción
Este es un proyecto sencillo diseñado para demostrar el dominio de etiquetas HTML semánticas y el flujo de trabajo con Git/GitHub.

### Características
- Uso de encabezados (`h1`, `h2`).
- Navegación mediante enlaces internos y externos.
- Estructura semántica (`header`, `main`, `footer`).
- Formulario de contacto básico.

### Despliegue
La página está publicada en línea gracias a **GitHub Pages**.
